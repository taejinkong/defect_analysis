# Watch Display Defect Analysis

Watch Display의 R/G/B/W 점등 이미지를 불러와 학습된 불량 데이터와 매칭하고, 결과를 대시보드로 시각화하는
**오프라인 단일 HTML 앱**이다.

설치, 서버, 인터넷 연결이 필요 없다. 이미지는 브라우저 안에서만 처리되며 네트워크로 전송되지 않는다.

## 현재 상태

**1~4단계 구현 완료.** 대시보드(5단계)는 아직 없다.

1단계 — 전처리:

- 원형 Active 영역 자동 검출 (Otsu → 최대 연결 성분 → 구멍 채우기 → 최소제곱 원 피팅)
- FPCB 방향 자동 추정 및 신뢰도 평가
- FPCB 6시 정렬 회전 보정 (사용자 확정 필수)
- 512×512 정규화 프레임 리샘플링 및 Active 마스크 생성
- 파일명 기반 패널 자동 그룹핑, 결손·중복·파싱 실패 처리
- 검출 결과 오버레이 UI, 수동 원 지정, 회전각 조정
- 합성 패널 생성기 — 실제 이미지 없이 파이프라인 검증용

2단계 — Rule 검출:

- 전역 강건 2차 곡면 배경 모델 (Otsu 사전 분할 + Tukey IRLS)
- 암점 면적 등급 판정 (小/中/大), 명점, 명선·암선 가로/세로
- 미점등 (전체/부분), 구동불량 의심 플래그
- 패턴 교차 확인 — 2개 이상 패턴에서 보여야 인정, 아니면 검수 대상
- 복수불량 판정 및 severity 기반 Primary Defect 선정
- 패널별 판정 근거(`decision_reason`)와 신뢰도 표시

임계값 튜닝:

- 13개 임계값 슬라이더. 값을 바꾸면 검출 박스가 즉시 갱신되는 실시간 미리보기
- 잘못된 조합(암점 中 상한 ≤ 小 상한 등)은 자동 보정하고 사유를 알림
- JSON 내보내기 / 가져오기, 기본값 복원, localStorage 자동 저장
- 임계값 변경 시 기존 분석 결과를 stale로 표시

3단계 — 라벨링 및 영속화:

- IndexedDB 영속화. 패널·이미지·라벨이 브라우저를 닫아도 유지됨
- 정규화 프레임 위에 점/선/박스로 불량 위치 라벨링
- 업로드 시 용도(학습용/분석용)·Model·Process·Equipment·작업자 메타데이터 입력
- 패널 승인 워크플로 (승인 시 라벨이 approved로 전환)
- 라벨 JSON 내보내기/가져오기 (이미지 픽셀 제외, 라벨·기하만)
- 이미지 Blob 저장, 재분석 없이 재실행 가능하도록 검출 결과 캐시

4단계 — kNN 유사도 검색 및 융합 판정:

- 패턴 이미지당 91차원 디스크립터(방사·각도 히스토그램 + 전역 통계), 4패턴 concat → 364차원 L2 정규화
- 정규화 프레임이 6시 정렬돼 있어 벡터가 회전 정렬된 위치 정보를 담음
- 승인된 학습 패널만 IndexedDB embedding store에 저장(feature_version 태깅)
- brute-force 코사인 유사도 검색, 유사도 가중 다수결
- Rule + kNN 융합: Rule이 불량을 판정하면 우선, 양품이면 kNN이 불량 승격 가능(구동불량 경로). 불일치는 검수 대상
- 학습 패널 수가 임계 미만이면 kNN 비활성, Rule 단독 판정

## 실행

```bash
npm install
npm run dev      # 개발 서버
npm test         # 175개 테스트
npm run build    # dist/index.html 단일 파일 생성 (~76KB)
```

빌드 산출물 `dist/index.html` 은 자기완결형이다. 더블클릭하면 브라우저에서 바로 열리고,
서버·인터넷·설치가 필요 없다.

## 개요

앱은 두 개의 독립된 기능 영역으로 구성된다.

- **학습**: 불량 Master 관리, 학습 이미지 등록, 위치 라벨링, 승인
- **분석**: 이미지 일괄 등록, 자동 불량 매칭, 검수, 대시보드

매칭 엔진은 결정론적 Rule 검출과 kNN 유사도 검색을 결합한다. 두 판정이 불일치하면 검수 큐로 올라가고,
사람의 수정이 곧 다음 학습 데이터가 된다. 딥러닝 프레임워크와 GPU를 쓰지 않는다.

## 임계값 튜닝 절차

1. 앱을 열고 실제 점등 이미지를 불러온다
2. 이미지를 클릭해 상세 화면을 연다 — 검출 박스와 `dark_area_ratio`가 표시된다
3. **임계값 조절**을 펼치고 슬라이더를 움직인다. 검출 결과가 즉시 갱신된다
4. 맞춰지면 **JSON 내보내기**로 저장한다
5. 다른 PC에서는 **JSON 가져오기**로 같은 설정을 재현한다

브라우저 저장소를 지우면 값이 사라진다. 내보낸 JSON이 유일한 백업이다.
